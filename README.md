# MyShelf

A lightweight macOS menubar app for quick, temporary file storage — built with Swift and AppKit.

Inspired by [Dropover](https://dropover.app), rebuilt from scratch as a personal learning project to explore macOS menubar app development.

## Features
- **Shake to summon**: shake your mouse to instantly bring up the shelf
- **Drag & drop**: drop files onto the shelf to store them temporarily
- **Dynamic resizing**: the shelf window resizes based on content

## Tech Stack
- Swift
- AppKit
- Developed with Xcode + Claude Code

## Status
Development complete. This is a personal study project and not affiliated with Dropover.
